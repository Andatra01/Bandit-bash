Ну готовьтесь  тут у меня знатно подгорело 13 уровень, реально 13....

1 - Создаём временную папку
mktemp -d (Создаёт уникальную временную папку в /tmp. Нужна потому что в домашней директории нет прав на создание файлов)

2 - Переходим в неё cd /tmp/tmp.EnLSki92lq

3 - cp ~/data.txt .

4 - Конвертируем hexdump в бинарный файл. Думал как обойти,  а ни как, ни разу этим не занимался, ужас какой....
xxd -r data.txt > data (xxd -r — переводит hexdump обратно в бинарный файл. Результат сохраняем в файл data)

Дальше начинается веселуха со слоями
5 - Проверяем тип файла file data (Показывает что за тип файла. Это главная команда которую повторяем после КАЖДОЙ распаковки)

6 - Слой 1 — gzip
mv data data.gz
gzip -d data.gz

(mv — переименовываем файл, даём правильное расширение .gz)

gzip -d — распаковываем gzip архив

7 - Слой 2 — bzip2
mv data data.bz2
bzip2 -d data.bz2

(Та же история, только теперь bzip2. Переименовываем и распаковываем)

8 - Слой 3 — gzip снова
mv data data.gz
gzip -d data.gz

(Да, снова gzip. Делаем то же самое)

9 - Слой 4 — tar
mv data data.tar
tar xf data.tar

tar xf — распаковываем tar архив.

(После распаковки появился файл data5.bin)

10 - Слой 5 — tar снова
bashtar xf data5.bin

Появился файл data6.bin

11 - Слой 6 — tar ещё раз
tar xf data6.bin

(Появился файл data8.bin)

12 - Слой 7 — gzip
mv data8.bin data8.gz
gzip -d data8.gz

Снова gzip, снова то же самое

13 - file data8

Выводит ASCII text — наконец-то текст! Вы уже читать устали, а я это вводил, а кто-то создавал. Страшные люди.
